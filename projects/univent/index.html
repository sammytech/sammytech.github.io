UNIVENT

DynamoDB, Node.js, Android SDK, Java

Goals

  • Students be aware of events on campus
  • Students knows which of their friends are attending events

Requirements for API and Mobile Application

  1. Authentication
  2. Register users
  3. Confirm users email
  4. Attend events
  5. Un-attend events
  6. Get trending events
  7. Retrieve events based on genre id
  8. Retrieve detailed event information including followings that are attending the event
  9. Retrieve followers
  10. Retrieve followings
  11. Add followings
  12. Delete followings
  13. Confirm followers
  14. Decline followers

Development

  • During the testing phases of this app, a web scrapper was developed in node.js to get all the events on the campus from the school's website and store them in our database.

  • Technologies

    • AWS EC2 – This is the web service we used to host our API.
    • Node.js – Used to build our API on because of its asynchronous I/O and its ability with the real-time applications.
    • Android SDK / Java – This is the language and software development kit necessary for creating android application
    • DynamoDB – This is a NoSQL database used to store all our data.
