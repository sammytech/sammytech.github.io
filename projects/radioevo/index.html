Goals

  • To work in a group that consists of foreign(French) students
  • To work with JavaFX
  • To work with a database(MySQL)
  • To consider internalization while building a software

Overview

Requirements

  1. Add a book to the system (Note: each book will have a barcode label pre-assigned – i.e., “stuck on” – to it. This will be a 5-digit barcode, with the first two digits indicating book type – e.g., “20” will be an English book, “30” a Math book, and so on).
  2. Modify information about a book in the system
  3. Delete a book from the system
  4. Add a student borrower to the system
  5. Modify information about a student borrower in the system (NOTE: Only “Administrator” workers can mark a student borrower as “delinquent”, or remove the “delinquency” mark)
  6. Delete a student borrower from the system
  7. Add a worker to the system (this functionality should include the feature to set the worker’s credentials – “Ordinary” or “Administrator”)
  8. Modify information about a worker in the system (this should allow the credentials to be changed – from “Administrator” to “Ordinary” and vice versa)
  9. Delete a worker from the system
  10. Check out (i.e, rent out) a book
  11. Check in (i.e., return) a book
  12. Run delinquency check – this will go over all books that are currently “late” and mark their associated borrower’s account as “delinquent”
  13. List all books that are currently checked out
  14. List all books that are currently available for check out
  15. List all student borrowers who have books checked out
