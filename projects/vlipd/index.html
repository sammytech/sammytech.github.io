VLIPD

Swift, Parse(MBaaS)

Goals

  • To allow users to share recorded videos
  • To allow users to collaborate on shared videos.

Requirements

  1. Register users using their emails
  2. Login users using their emails and password
  3. Record videos
  4. Switch between front and back camera for recording
  5. List of people that a video is being shared with
  6. Pick a maximum of 5 users that user wants to share video with

Design

Library Main View

Development

  • This project was made with no budget; therefore, we took advantages of the free platforms which made us focus on the development of the application. This application was made for iOS only and we were able to demonstrate it at Imagine RIT.

  • Technologies

    • Swift – This is an MBaaS which allowed us as a team to worry less about the backend functionality of the application.
    • Parse – This is an MBaaS which allowed us as a team to worry less about the backend functionality of the application.
  • All user interfaces were coded instead of using the interface builder because I felt I had more control on what happens and what is being displayed in the app.
